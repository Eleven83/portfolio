CSS = Casscading Style Sheets. A key feature of CSS is what's known as "inheritance". This means it relies on an ancestor-descendent relationship. Each nested element receives its' parent color, font, etc...

You can have several CSS files (or sheets) that will be referenced in HTML and all of them will be used to find out the final and visible style for your page. Think of it in terms of laws. With CSS, the most specific rule is applied to every element That's where the word cascading comes from in CSS.This requires Abstract Thinking and the ability to find similarities.

What is CSS?

CSS is code written to control the STYLE of html elements.

How does CSS control html?

If, for example, you want to make all the <h1> have a black background with white text, you would write: 'h1 { background-color: black; color: white;
}'


h1 is called a "selector" and it tells the browser to apply the following rules to every h1 element. Everything between the curly braces is a rule.

In CSS you first write a selector. This defines what elements of a page this style will apply to. You can select an html tag and that rule will apply to the same tags on the page.

There are 3 ways to include CSS Styling:

  1. Write CSS in the <head> of hte html. This is only good for small projects.
  2. Link your html to a separate CSS file
    • Helps stay organized on larger projects
  4. To do this you would write all the structural html in one file and call it main.html or similar. In a separate file, put all the CSS code and call that main.css or similar. In the head element of the html document you add a link to your main.css. EX: <link rel="stylesheet" href="main.css">
  5. Write the style inline with html
    • This is generally a bad idea because there is lots of repeated code
  7. With this method, you modify the style attributes of every individul element.

4 Main Points of Box Sizing

  1. HTML elements are boxes and each box has 4 components.
  2. Because there are so many components to each box it can be hard to get the size of the box just right.
  3. There are 2 techniques you can use to help deal with sizing issues:
    1. Set the size in % rather than px
    2. Set the box-sizing attribute to border-box for every element
  5. Different browsers work slightly differently. Sometimes this causes different browsers to display the same code differently

Box Positioning

  1. DIVs are block elements so, by default, they take up the entire width of a page.
  2. Adding the rule display: flex; to the appropreaite CSS will override this behavior and let DIVs appear next to eachother.
